## 🚀 Tecnologías Utilizadas

Este proyecto fue construido con algunas de las mejores herramientas modernas del ecosistema de desarrollo web:

* **[React](https://react.dev/)** - Biblioteca principal para el desarrollo de la interfaz de usuario.
* **[Vite](https://vitejs.dev/)** - Entorno de desarrollo y empaquetador ultrarrápido.
* **[TypeScript](https://www.typescriptlang.org/)** - Tipado estático para un código más robusto y sin errores.
* **[Tailwind CSS](https://tailwindcss.com/)** - Framework de utilidades CSS para un diseño a medida y altamente fluido.
* **[React Icons](https://react-icons.github.io/react-icons/)** - Iconografía moderna y ligera.

## 🌟 Características

- 🌓 **Modo Oscuro/Claro:** Soporte integrado para que la interfaz se adapte a tus preferencias.
- 📱 **Diseño 100% Responsivo:** Completamente adaptable a teléfonos móviles, tablets y computadoras de escritorio.
- 🎨 **Animaciones Fluidas:** Transiciones suaves que mejoran de forma sutil la experiencia de usuario (Hover, Intersect Observers, etc).
- ⚡ **Alto Rendimiento:** Optimización y recargas instantáneas gracias al servidor de Vite.

## 💻 Instalación y Uso Local

Para probar o continuar trabajando en este proyecto de forma local, sigue estos sencillos pasos:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/nicolasgrajaleshoyos/portafolio.git
   cd portafolio
   ```

2. **Instalar las dependencias:**
   Puedes utilizar `npm`, `yarn` o `bun`:
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Visualizar el sitio:**
   Abre tu navegador preferido e ingresa a `http://localhost:5173`
